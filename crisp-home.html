Welcome to Crisp

Crisp is a personal open-source project created with an intent to showcase reference implementations and appearances of various rendering techniques I've encountered throughout studying and independent exploration of topics in computer graphics.

The application includes algorithms and topics from both real-time and offline rendering. In addition, I'm developing a custom GUI layer for learning purposes. The basic screens of the application are shown below.

The real-time rendering part of the project is built on top of Vulkan API. The ray tracer subsystem is inspired by the architecture of Nori, an educational ray tracer used in the Computer Graphics AS15 course at ETH Zurich.

The project's source code is hosted on my GitHub profile.
