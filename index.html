Crisp is a personal open-source project created with an intent to showcase reference implementations and appearances of various rendering techniques I've encountered throughout studying and independent exploration of topics in computer graphics.

The application includes algorithms and topics from both real-time and offline rendering. The real-time rendering part of the project is built on top of Vulkan API. The ray tracer subsystem is inspired by the architecture of Nori, an educational ray tracer used in the Computer Graphics AS15 course at ETH Zurich.

Its source code is hosted on GitHub over here.

The shader ball was modeled by myself in Blender with inspirations from various other shader balls used in other rendering systems.

The environment maps in the showcased images produced with ray tracing were provided by Bernhard Vogl and Paul Debevec.
